# Dashboard

The Dashboard provides a real-time, visual interface for the Kubernetes Auto-Scaling Intelligence Engine.

## Features
- Built with React, Vite, and Recharts.
- Provides real-time monitoring of scaling events, cost savings, and SLA compliance.
- Supports light and dark mode themes.
- Visualizes traffic predictions against actual traffic.

## Running Locally

Make sure you have Node.js installed.

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:5173`.
